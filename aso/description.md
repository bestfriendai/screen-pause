# ScreenPause - Focus Timer & Accountability

ScreenPause helps you stay focused and beat phone addiction. Use the Pomodoro technique, track your progress, and challenge friends to build healthy screen time habits.

## Features
- Focus timer with customizable durations
- Category-based sessions (Work, Study, Reading, etc.)
- Track daily, weekly, monthly progress
- Build focus streaks
- Challenge friends
- Leaderboards and achievements

## How It Works
1. Choose a duration (15, 25, 45, 60 min)
2. Select a category
3. Start focusing
4. Track your progress

Perfect for students, professionals, and anyone wanting to be more productive.
